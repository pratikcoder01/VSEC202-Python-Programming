# VSEC202 Python Programming – Practical Solutions

This repository contains beginner-friendly Python programs covering the full practical syllabus for **VSEC202 – Python Programming**.

## Structure

- Module 01: Basics (inputs, operators, simple calculations)
- Module 02: Data Structures (lists, tuples, dictionaries)
- Module 03: Control Flow & Functions
- Module 04: File Handling & Packages
- Module 05: Exceptions, Logging, Debugging
- Module 06: Object-Oriented Programs
- Module 07: GUI Programs (Tkinter)
- Module 08: Regular Expressions
- Module 09: NumPy Basics
- Module 10: Pandas & Matplotlib Basics

All programs are written in a **simple, beginner-friendly style** with:

- user input using `input()`
- clear variable names
- step-by-step comments
- sample outputs in comments

You can run any program like this:

```bash
python filename.py
```

Feel free to use this as a reference for learning or for your laboratory practice.
